this directory's source code is a sentence separating app,
if you want to run it ,compile and copy app-separ.exe to directory ./app,
goto ./app to run it,
usage: ./app-separ.exe myfile.txt , the result is in file myfile.out2.txt
in myfile.out2.txt, lines start with '$1,,' are first step's result,
lines start with '$3,,' are final result.


这个目录的源程序是一个语句分解app,编译并拷贝到./app目录，移动到./app目录
去运行，用法：./app_separ.exe myfile.txt ，结果在myfile.out2.txt文件里。
用"$1,,"开始的行是第一步结果,用"$3,,"开始的行是第二步结果,第二步结果是 最后结果.
